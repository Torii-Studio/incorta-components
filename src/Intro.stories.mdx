# Get started

Incorta Component System is a reusable component library made for Incorta products.

## Install

Incorta Component System components are written in React, and its stories are written in [Component Story Format](https://medium.com/storybookjs/component-story-format-66f4c32366df). It requires Storybook version 5.2-beta and up.

Add Incorta Design System to your project.

`npm install --save @incorta/component-system`
